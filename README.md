# Kompilacja - GNU/Linux
Sklonuj repozytorium, przejdź do niego, i użyj make:
```
make -j$(nproc) all clean
```
Lub, jeżeli chcesz poczekać dłużej i nie chcesz sprzątać po kompilacji, po
prostu `make`
